## 1. Создание venv

### В папке с проектом:

```bash
py -m venv venv
```

- `venv` — имя папки с окружением.
---

## 2. Активация окружения

### Через **PowerShell**

```powershell
.venv\Scripts\Activate.ps1
```

## 3. Установка зависимостей

```bash
pip install -r requirements.txt
```
